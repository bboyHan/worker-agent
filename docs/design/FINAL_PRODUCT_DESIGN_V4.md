# Worker Agent · v4.0 一页纸执行摘要

## 一句话价值
> "打开Worker Agent — 你的电脑就是一个统一的操作系统。任意页面随时呼出AI对话，窗口自由拖拽分割，像桌面OS一样操控你的Agent和工作。"

## V4.0 三大核心更新

### 1. 全局浮动Chat面板（4级状态）
- 右下角FAB按钮呼出 / 快捷键 ⌘+Shift+K
- 四级状态：气泡(48px) → 半展开(280px) → 展开(400px) → 最大化(600px)
- 跨页面共享会话，关闭位置记住，固定后不随页面切换消失
- 通知badge提醒未读，拖拽边界碰撞检测

### 2. 任意拖拽分割窗口
- 拖拽分割线调整面板比例（15%~85%），实时比例显示
- 拖到边缘吸附（Snap Layouts，6个布局热力图）
- 拖入中间拆分面板（1窗口→3面板）
- 布局快照保存/恢复，localStorage持久化

## 设计原则（新增）
1. **默认极简**：只有搜索框+最近任务，新Mac桌面感
2. **随时对话**：Chat面板永远可呼出，不中断当前工作
3. **自由布局**：窗口大小/位置完全由用户掌控
4. **键盘优先**：80%功能键盘完成
5. **状态持久**：所有布局/面板状态自动保存恢复
6. **渐进复杂度**：每层只展示当前需要的东西
