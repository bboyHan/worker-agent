# Worker Agent v4.0 执行摘要（更新版）

## 核心宗旨
打造方便你自己的一站式工作平台 → 全面管理 Hermes + 随时随地远程对话分派 Agent 工作

## 五大核心板块

1. **Agent 管理** — 状态/能力/模型全面管控
2. **会话管理** — 列表/切换/搜索
3. **右侧 Chat 抽屉**（核心 UI）— 三态切换：折叠 5px / 小窗 380px / 全宽 50%
4. **类 VS Code 分屏 + 标签页** — 最多 4 面板，拖拽调整比例
5. **远程控制** — 企微/飞书/微信远程对话分派 Agent 工作
6. **Agent 运行监控** — 查看 Agent 正在做什么的分析视图
