## 一、设计目标

| 设计决策 | 目标 |
|----------|------|
| 布局原则 | VS Code 级极简，最大化工作区，最小化导航占地 |
| 导航方式 | 图标栏（窄屏仅图标，悬停显示名称）+ 顶部全局搜索 |
| 交互模式 | Agent 聊天式交互（Thought → Code Change → Product Summary） |
| CLI 整合 | 可拖拽分界线的浮动面板，左下角一键折叠/展开 |
| 视觉风格 | 暗色主题，高对比度，无装饰 |

## 三、四大核心区域

### 3.1 顶部全局搜索栏（居中）

- **显示位置**: 页面最顶部，水平居中
- **触发方式**: Ctrl+P / Cmd+P / 点击
- **样式**: 输入框，placeholder "搜索命令、文件、Agent..."
- **功能**: Raycast/Ctrl+P 风格快速搜索

### 3.2 左侧图标栏（50px 窄栏）

| 图标 | 功能 | 悬停显示 |
|------|------|----------|
| 🗒 | 文件管理 | "文件" |
| 📋 | 任务看板 | "任务" |
| 🤖 | Agent 控制台 | "Agent" |
| ⚙️ | 工作流编排 | "工作流" |
| 🖥️ | 服务器监控 | "服务器" |
| 📚 | 知识库 | "知识库" |
| 🔌 | 插件管理 | "插件" |
| ⚡ | 全局搜索 | "全局搜索" |
| ⚙ | 设置 | "设置" |
| ⇕ | 折叠 CLI | "收起面板" |

**交互规则:**
- 默认仅显示图标（无文字）
- 鼠标悬停 300ms 后显示 tooltip（菜单名 + 快捷键）
- 当前激活的图标高亮（左侧有竖线标识）
- 图标栏宽度固定 50px，不可拉伸

### 3.3 可拖拽 CLI 面板

- **位置**: 固定在右侧面板（默认宽度占整体 40%-50%），可通过拖拽分界线调整
- **分界线**: 带微光动画的细线（2px），拖拽时加粗为 4px
- **折叠按钮**: 左下角，圆形按钮，点击后 CLI 面板完全隐藏，工作区全屏
- **展开方式**: 点击左下角 "⇕" 按钮或 Ctrl+` 快捷键

**Agent 聊天交互区（CLI 面板内部）:**

参考 Trae SOLO 的设计模式，每个任务完成后的交互流包含：

1. **Thinking（思考过程）** — 可折叠的 AI 思考链
2. **Code Changes（代码变更）** — 显示文件修改列表，支持一键查看 diff
3. **Product Summary（产物汇总）** — 任务完成后的产物清单

## 五、交互协议规范

### 5.1 折叠/展开动画

| 操作 | 动画 |
|------|------|
| 折叠 CLI | 面板以 200ms ease-out 向右滑动隐藏 |
| 展开 CLI | 面板以 200ms ease-in 从右向左滑入 |
| 折叠动画期间 | 工作区 content 区域平滑扩展填补空间 |

### 5.2 拖拽分界线

| 状态 | 行为 |
|------|------|
| 正常 | 2px 细线，hover 时变为 4px，cursor: col-resize |
| 拖拽中 | 4px 半透明蓝色高亮线，拖拽时锁定光标位置 |
| 拖拽停止 | 面板位置锁定，显示宽度百分比 |

## 六、Agent 交互模式详解（参考 Trae SOLO）

### 6.1 Thought Process（思考过程）

- 在 Agent 面板中以灰色框显示
- 支持折叠/展开，默认折叠（减少视觉杂乱）
- 可勾选 "Always show Thought process" 选项

### 6.2 Code Changes（代码变更）

- 文件列表 + 变更统计 (+N/-N)
- 每行支持 "查看变更" 按钮（展开 inline diff）
- 底部有 "全部撤销" / "全部保留" 批量操作

### 6.3 Product Summary（产物汇总）

- 任务完成时自动显示
- 绿色对勾表示"任务完成"
- 点击可展开详细产物清单
- 支持 "复制产物列表" 快捷键

## 七、设计优势

| 维度 | 优势 |
|------|------|
| 屏幕空间 | 图标栏仅 50px，最大化工作区 |
| 导航效率 | 悬停 tooltip + 全局搜索，双通道导航 |
| CLI 灵活性 | 可拖拽 + 可折叠，按需展开 |
| 交互一致性 | Thought → Code Change → Summary 统一模式 |
| 视觉清爽 | 暗色极简，无多余装饰，高信息密度 |
| 可扩展性 | 图标栏可插拔，新增功能只需加图标 |

## 八、技术实现要点

| 组件 | 技术 |
|------|------|
| 图标栏 | Flexbox + CSS Tooltip + JavaScript 悬停检测 |
| 全局搜索 | Ctrl+P 事件监听 + 虚拟滚动列表 |
| 拖拽分界线 | mousedown/mousemove/mouseup 事件 + cursor 样式 |
| CLI 折叠 | CSS transform + transition 动画 |
| Agent 聊天 | 消息气泡组件 + diff 渲染库 |
| 主题切换 | CSS 变量 + dark mode media query |
