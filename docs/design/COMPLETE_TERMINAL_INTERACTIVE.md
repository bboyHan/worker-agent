# Worker Agent · 终端交互设计规格 v2.0

> **设计原则**：所有复杂都藏在简单之后。终端是入口，不是终点。

---

## 一、CLI 命令体系

### 1.1 主命令

```
worker-agent    ← 平台核心命令
├── start       启动平台
├── stop        停止平台
├── status      查看平台状态
├── config      配置管理
└── help        帮助
```

### 1.2 命令分组

#### 任务管理
```
worker-agent task
├── create      创建任务
├── list        列出任务
├── done        完成任务
├── delete      删除任务
├── search      搜索任务
└── board       看板视图
```

#### Agent 管理
```
worker-agent agent
├── list        列出所有Agent
├── start       启动Agent
├── stop        停止Agent
├── restart     重启Agent
├── status      查看Agent状态
├── logs        查看Agent日志
└── config      配置Agent
```

#### 文件管理
```
worker-agent file
├── ls          列出文件
├── tree        文件树
├── search      搜索文件
├── open        打开文件
├── edit        编辑文件
├── delete      删除文件
└── sync        同步文件
```

#### 代码管理
```
worker-agent code
├── edit        编辑代码
├── review      代码审查
├── generate    生成代码
├── test        运行测试
└── commit      提交代码
```

#### 部署管理
```
worker-agent deploy
├── docker      Docker部署
├── k8s         K8s部署
├── vercel      Vercel部署
├── status      部署状态
└── rollback    回滚部署
```

#### 监控告警
```
worker-agent monitor
├── server      服务器监控
├── alert       告警管理
├── logs        查看日志
└── graph       可视化
```

#### 知识管理
```
worker-agent knowledge
├── list        列出知识库
├── create      创建文档
├── search      搜索
└── add         添加知识
```
