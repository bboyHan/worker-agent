## 六、ASCII 原型交互说明

### 6.1 各区域交互方式

| 区域 | 交互方式 |
|------|------|
| 图标栏 | `Enter` 选中、`Enter` 执行 |
| 搜索栏 | `↑↓` 导航、`Enter` 执行 |
| 文件树 | `Enter` 打开文件夹、`←→` 展开/收起 |
| 任务看板 | `Enter` 查看详情、`Del` 删除任务 |
| Agent 控制台 | `Space` 展开详情、`Enter` 操作 |
| 服务器列表 | `Enter` 查看详情、`S` SSH 连接 |
