# Worker Agent v4.0 — 全景功能设计书

## 一、核心定位

打造方便你自己的一站式工作平台：
- ✅ 全面管理 Hermes Agent（界面化深度操作）
- ✅ 随时随地通过企微/飞书/微信远程对话分派 Agent 工作
- ✅ 查看所有 Agent 运行中的分析视图
- ❌ 不需要在 app 里写代码

## 二、Chat 右侧抽屉（核心 UI 组件）

### 2.1 三态设计

| 状态 | 宽度 | 行为 |
|------|------|------|
| 折叠态 | 5px | 仅显示右侧边缘条，可点击展开 |
| 小窗态 | 380px | 默认宽度，可拖拽 200-800px |
| 全宽态 | 50% 宽 | 默认占 50%，可拖拽调整左右比例 |

### 2.2 交互

```
折叠态 ←→ 小窗态 ←→ 全宽态
  ↑           ↑          ↑
  click     double     drag edge
```

- 所有面板边界支持拖拽调整
- 有最小宽度/高度限制
- 拖拽时显示辅助线

### 2.3 CSS 实现

```css
/* 折叠态：5px */
.chat-drawer.state-folded {
  width: 5px;
  cursor: col-resize;
  border-left: 1px solid var(--brd);
}

/* 小窗态：380px */
.chat-drawer.state-minimized {
  width: 380px;
  min-width: 200px;
  max-width: 800px;
}

/* 全宽态：50% */
.chat-drawer.state-expanded {
  width: 50%;
}

/* 可拖拽边界 */
.chat-resize-handle {
  position: absolute;
  left: 0;
  top: 0;
  bottom: 0;
  width: 6px;
  cursor: col-resize;
  border-left: 2px solid transparent;
  transition: border-color .15s;
}
.chat-resize-handle:hover,
.chat-resize-handle.dragging {
  border-left-color: var(--ac);
  background: rgba(124,58,237,.08);
}
```

## 三、类 VS Code 分屏 + 标签页系统

### 3.1 多面板分屏（最多 4 面板）

```
┌──────┬──────┐
│ Panel│ Panel│
│  1   │  2   │
├──────┼──────┤
│ Panel│ Panel│
│  3   │  4   │
└──────┴──────┘

布局数据结构：
Layout {
  type: "grid" | "horizontal" | "vertical"
  children: Panel[]
  ratios: number[]  // 各面板比例
}
```

### 3.2 拖拽调整

- 所有面板边界支持拖拽
- 显示辅助线（对齐到 25%/50%/75%）
- 鼠标拖拽更新 CSS flex 比例

### 3.3 标签页系统

```
┌──────┬──────┬──────┐
│ Tab1 │ Tab2 │ Tab3 │ x
├──────┴──────┴──────┤
│  Panel Content     │
└────────────────────┘
```

- 每个面板可含多个标签页
- 拖拽标签页到其他面板
- 右键菜单：固定 / 关闭其他 / 关闭右侧
- 标签页可关闭（x按钮）

## 四、远程控制（远程对话分派）

### 4.1 核心流程

```
外出时 ← 企微/飞书/微信 ← 发指令给 Agent ← 分派项目/任务/工作
  ↑                                    ↑
监控                                    对话
```

### 4.2 功能设计

| 功能 | 说明 |
|------|------|
| IM 远程链接 | 企微/飞书/微信 三个入口 |
| 对话分派 | 从 IM 直接分派 Agent 工作 |
| 任务监控 | 查看 Agent 运行状态 |
| 消息通知 | Agent 完成任务后推送通知 |

### 4.3 架构图

```
┌──────┐  企微/   ┌─────────────┐  消息  ┌────────┐
│ 手机  ├────IM────│  Worker     ├───────│  Agent │
│ 远程  │         │  Messenger  │        │  执行  │
└──────┘         └─────────────┘        └────────┘
                        ↑
                   状态回传
```
